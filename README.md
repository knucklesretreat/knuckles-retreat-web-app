# Knuckles Retreat Official Website

Live site: [https://www.knucklesretreat.com](https://www.knucklesretreat.com)

Built with [Qwik](https://qwik.builder.io/) + [Qwik City](https://qwik.builder.io/qwikcity/overview/),
server-side rendered and deployed to **Vercel Edge Functions**.

## Tech Stack

| Area        | Tooling                                             |
| ----------- | --------------------------------------------------- |
| Framework   | Qwik 1.16 / Qwik City                               |
| Build tool  | Vite 6                                              |
| Language    | TypeScript 5                                        |
| Hosting     | Vercel Edge Functions (`adapters/vercel-edge`)      |
| Backend API | Firebase Cloud Functions                            |
